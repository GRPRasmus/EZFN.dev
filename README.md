Language: Englisch
Step 1. You have to go to the ezfn.dev folder in the contents directory of epic here C: \ Program Files \ Epic Games
Step 2: You have to push the folder by drag and drop and go to the folder ezfn.dev and start the EzfnLauncher.exe and enter your data from your ezfn account
